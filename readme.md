# notadd

Notadd 是基于 Laravel 5.3 + Vue.js 的开源 CMS、商城、论坛、博客、问答、微信...
可以是任何你想要的，我们将其分为模块（实现大功能，诸如用户系统、商城系统、CMS 系统）、插件（增强模块功能）、模板（改变前台界面）。

## 系统环境

操作系统： Linux（推荐）/Mac OS/ Windows 2008+

PHP版本 ： 7.0+

数据库： PostgreSQL（推荐）/MariaDB/MySQL/SQLite3
